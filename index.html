🎓 Teacherize 가이드

Teacherize로 수업 준비를 더 쉽고 빠르게 해보세요!

1. Teacherize 시작하기

  1. 구글 스프레드시트 열기
  2. '확장 프로그램' → '추가 기능 가져오기' → 'Teacherize' 검색 및 설치
  3. '확장 프로그램' → 'Teacherize' → 'Set Properties'에서 API 키 입력 및 설정

2. Teacherize 사용하기

  1. 스프레드시트에 예시 데이터 입력:
    • A5, B5, C5에 머리글 (예: 날짜, 행사명, 활동내용)
    • A6, B6, C6에 예시 데이터
  2. 새 데이터 입력: A7, B7에 새로운 정보 입력
  3. C7 셀에 Teacherize 함수 사용:

    =teacherize($A$6:$B$6,$C$6,A7:B7)

  4. Enter 키를 눌러 결과 확인

3. 유용한 팁

  • 다양한 예시를 입력할수록 더 좋은 결과를 얻을 수 있어요.
  • 'Apply to Selected Cells' 버튼으로 여러 셀에 한 번에 적용 가능
  • 'Paste Values to All'로 함수 결과를 고정값으로 변경 가능

🎓 Teacherize Guide

Make your lesson preparation easier and faster with Teacherize!

1. Getting Started with Teacherize

  1. Open Google Spreadsheets
  2. 'Extensions' → 'Get add-ons' → Search and install 'Teacherize'
  3. 'Extensions' → 'Teacherize' → 'Set Properties' to enter API key and settings

2. Using Teacherize

  1. Enter example data in the spreadsheet:
    • A5, B5, C5 for headers (e.g., Date, Event Name, Activity Details)
    • A6, B6, C6 for example data
  2. Enter new data: Input new information in A7, B7
  3. Use Teacherize function in C7:

    =teacherize($A$6:$B$6,$C$6,A7:B7)

  4. Press Enter to see the result

3. Useful Tips

  • More diverse examples lead to better results.
  • Use 'Apply to Selected Cells' to apply to multiple cells at once
  • 'Paste Values to All' changes function results to fixed values

Prepare creative and efficient lessons with Teacherize! 🚀📚
