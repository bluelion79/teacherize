🎓 Teacherize 간편 매뉴얼

안녕하세요, 선생님! Teacherize는 여러분의 수업 준비를 도와주는 특별한 도구예요. 이 간편 매뉴얼과 함께 Teacherize의 사용법을 쉽게 배워볼까요?

1. Teacherize 시작하기

Step 1: Teacherize 설치하기

  1. 구글 스프레드시트를 엽니다.
  2. 상단 메뉴에서 '확장 프로그램' → '추가 기능 가져오기'를 클릭해요.
  3. 검색창에 'Teacherize'를 입력하고 설치 버튼을 눌러주세요.

Step 2: Teacherize 설정하기

  1. 스프레드시트 상단 메뉴에서 '확장 프로그램' → 'Teacherize' → 'Set Properties'를 클릭해요.
  2. 오른쪽에 나타나는 창에서 API 키를 입력해주세요. (API 키는 선생님의 학교에서 제공해줄 거예요!)
  3. 사용할 AI 모델을 선택하고, 다른 설정들도 원하는 대로 조절해보세요.
  4. 설정이 끝나면 'Save Settings' 버튼을 눌러주세요.

2. Teacherize 사용하기

Step 3: 예시 만들기

  1. 스프레드시트에 다음과 같이 예시를 입력해봐요:
    • A5에는 '날짜', B5에는 '행사명', C5에는 '활동내용'이라고 머리글을 작성해요.
    • A6, B6, C6에 각각 예시 문장을 작성해주세요.
  2. 이렇게 예시를 만들어주면, Teacherize가 이를 바탕으로 학습해요.

Step 4: Teacherize 함수 사용하기

  1. A7, B7에 새로운 내용을 입력해주세요.
  2. C7 셀을 선택하고, 함수 입력창에 다음과 같이 입력해보세요:

=teacherize($A$6:$B$6,$C$6,A7:B7)

여기서 $A$6:$B$6는 예시 질문들, $C$6는 예시 답변, A7:B7은 새로운 질문이에요.

Step 5: 결과 확인하기

  1. Enter 키를 누르면, Teacherize가 마법처럼 새로운 답변을 C7 셀에 만들어줄 거예요!
  2. 이 답변을 바탕으로 수업 자료를 만들어보세요.

3. 꿀팁과 주의사항

💡 꿀팁

  • 다양한 예시를 많이 입력할수록 더 좋은 결과를 얻을 수 있어요.
  • 'Apply to Selected Cells' 버튼을 사용하면 여러 셀에 한 번에 Teacherize를 적용할 수 있어요.
  • 'Paste Values to All' 버튼으로 모든 Teacherize 함수의 결과를 고정된 값으로 바꿀 수 있어요.

⚠️ 주의사항

  • API 키는 비밀번호처럼 소중히 다뤄주세요. 다른 사람에게 알려주면 안 돼요!
  • Teacherize가 만든 내용은 항상 선생님이 검토해주세요. 가끔은 이상한 답변을 할 수도 있거든요.
  • 학생들의 개인정보가 포함된 내용은 입력하지 말아주세요.

이제 Teacherize를 사용할 준비가 됐어요! 어려운 점이 있다면 언제든 도움을 요청해주세요. 즐거운 수업 준비 되세요! 🚀📚

🎓 Teacherize Simple Manual

Hello, teachers! Teacherize is a special tool to help you prepare for your classes. Let's learn how to use Teacherize easily with this simple manual!

1. Getting Started with Teacherize

Step 1: Installing Teacherize

  1. Open Google Spreadsheets.
  2. Click on 'Extensions' → 'Get add-ons' in the top menu.
  3. Type 'Teacherize' in the search bar and click the install button.

Step 2: Setting up Teacherize

  1. In the spreadsheet's top menu, click 'Extensions' → 'Teacherize' → 'Set Properties'.
  2. Enter your API key in the window that appears on the right. (Your school should provide you with the API key!)
  3. Select the AI model you want to use and adjust other settings as desired.
  4. When you're done, click the 'Save Settings' button.

2. Using Teacherize

Step 3: Creating Examples

  1. Enter examples in the spreadsheet like this:
    • Write 'Date' in A5, 'Event Name' in B5, and 'Activity Details' in C5 as headers.
    • Write example sentences in A6, B6, and C6 respectively.
  2. By creating these examples, Teacherize will learn from them.

Step 4: Using the Teacherize Function

  1. Enter new content in A7 and B7.
  2. Select cell C7 and enter the following in the function input bar:

=teacherize($A$6:$B$6,$C$6,A7:B7)

Here, $A$6:$B$6 are the example questions, $C$6 is the example answer, and A7:B7 are the new questions.

Step 5: Checking the Results

  1. Press Enter, and Teacherize will magically create a new answer in cell C7!
  2. Use this answer to create your teaching materials.

3. Tips and Precautions

💡 Tips

  • The more diverse examples you input, the better results you can get.
  • Use the 'Apply to Selected Cells' button to apply Teacherize to multiple cells at once.
  • The 'Paste Values to All' button can change all Teacherize function results to fixed values.

⚠️ Precautions

  • Treat your API key like a password. Don't share it with others!
  • Always review the content created by Teacherize. It might occasionally give strange answers.
  • Do not input any content that includes students' personal information.

Now you're ready to use Teacherize! If you have any difficulties, feel free to ask for help. Happy lesson planning! 🚀📚
